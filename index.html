8 thư viện React mạnh mẽ để thử vào năm 2021

28 tháng 06, 2021 - 1258 lượt xem

Sử dụng các thư viện phù hợp để tối ưu hóa quy trình làm việc của dự án

1. react-select

Một sự thay thế tuyệt vời cho Component trong React.js

Trình bày một cách hoàn toàn mới để phát triển các thành phần React.js mạnh mẽ

Các tính năng nổi bật
  • Cách tiếp cận dữ liệu linh hoạt, với các chức năng có thể tùy chỉnh
  • API tạo kiểu có thể mở rộng Emotion (Một số thư viện javascrpit được thiết kế cho kiểu CSS viết)
  • API chền thành phần để kiểm soát hoàn toàn đối với hành vi giao diện người dùng
  • Nhóm tùy chọn, hỗ trợ cổng thông tin, hoạt ảnh và hơn thế nữa

Để cài đặt:

npm i react-select

2. react-dnd

Drap and Drop for React

Một tập hợp các tiện ích React để giúp bạn xây dựng giao diện kéo và thả phức tạp trong khi vẫn tách rời các thành phần

Các tính năng nổi bật
  1. Hoàn hảo cho các ứng dụng
  2. Được xây dựng dựa trên API kéo thả và HTML5.
  3. React DnD sử dụng dữ liệu chứ không phải xem quảng cáo là nguồn sự thật.

Để cài đặt:

npm i react-dnd
